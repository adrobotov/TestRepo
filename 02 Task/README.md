# Решение второй задачи
Решение неоптимально по скорости вычисления, как вариант ускорения можно предварительно сформировать таблицу из одного столбца содержащую все вычисленные значения чисел фибоначчи на диапазоне допустимых значений, а в тестирующей функции осуществлять поиск проверяемого значения на присутствие в ней, но это требует затрат по занимаемой долговременной памяти. 
Второй файл факультативный, сгенерирован с использованием Libre Office и Notepad++, использовался для тестирования функции.
